# 🎲 Dice Duel - Tkinter Dice Rolling Game

Dice Duel is a simple and fun Python GUI application that simulates rolling a pair of dice. Built using Tkinter and PIL (Pillow), this app randomly displays dice face images every time you click the **ROLL** button, mimicking the experience of a physical dice roll.

---

## 🖥️ Features

- Simple and intuitive GUI using **Tkinter**
- Randomized dice rolls using **Python's `random` module**
- Dynamic image display with **Pillow (PIL)**
- Fun visual dice roll experience
